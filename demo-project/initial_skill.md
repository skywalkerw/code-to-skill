# Apache Fineract 会计凭证 Skill

> 版本: 0.1.0-minimal-baseline
> 适用: 用户给出 Fineract 业务交易，输出会计凭证或澄清说明

## 任务

根据用户描述和可用代码上下文，判断交易是否足以生成 Fineract 会计凭证。信息充分时输出「## 会计凭证」；信息不足时先澄清。

## 科目对（现金制）

### 科目对（现金制）

- 先查相关 Fineract 会计处理代码，再选择借方和贷方科目。
- 不确定具体科目时使用业务含义说明，不要编造不存在的科目编码。

## 输出要求

### 输出要求

- 使用 Markdown 表格，至少包含摘要、借方、贷方、金额。
- 金额必须来自用户输入。
- 末尾写明借贷是否平衡。

## 约束

- 无金额、关键拆分缺失、或会计期间关闭时，不要生成凭证，改为要求补充信息。
- 借贷不平衡时不得输出最终凭证。
